JSONPath

JSONPath predicates allow you to select specific values from JSON documents in request fields. This is useful when you need to match requests based on specific JSON content.

Usage

Use the jsonpath parameter on any predicate to scope the match to selected JSON content:

{
  "predicates": [{
    "equals": { "body": "test-value" },
    "jsonpath": {
      "selector": "$.item.name"
    }
  }],
  "responses": [{
    "is": { "body": "Matched!" }
  }]
}

JSONPath Syntax

Expression Description
$ Root object
.property Child property
..property Recursive descent
[n] Array index
[*] All array elements

Example

Given this request body:

{
  "order": {
    "items": [
      { "name": "Widget", "quantity": 5 },
      { "name": "Gadget", "quantity": 3 }
    ]
  }
}

The following predicates would match:

// Match first item name
{
  "predicates": [{
    "equals": { "body": "Widget" },
    "jsonpath": {
      "selector": "$.order.items[0].name"
    }
  }]
}

// Match any item with name "Gadget"
{
  "predicates": [{
    "equals": { "body": "Gadget" },
    "jsonpath": {
      "selector": "$.order.items[*].name"
    }
  }]
}
