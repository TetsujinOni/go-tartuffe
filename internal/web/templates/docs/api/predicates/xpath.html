XPath

XPath predicates allow you to select specific values from XML documents in request fields. This is useful when you need to match requests based on specific XML content.

Usage

Use the xpath parameter on any predicate to scope the match to selected XML content:

{
  "predicates": [{
    "equals": { "body": "test-value" },
    "xpath": {
      "selector": "//item/name"
    }
  }],
  "responses": [{
    "is": { "body": "Matched!" }
  }]
}

Namespaces

When dealing with namespaced XML, use the ns object to define namespace prefixes:

{
  "predicates": [{
    "equals": { "body": "test-value" },
    "xpath": {
      "selector": "//ns:item/ns:name",
      "ns": {
        "ns": "http://example.com/namespace"
      }
    }
  }],
  "responses": [{
    "is": { "body": "Matched!" }
  }]
}

Example

Given this request body:

<order>
  <item>
    <name>Widget</name>
    <quantity>5</quantity>
  </item>
</order>

The following predicate would match:

{
  "predicates": [{
    "equals": { "body": "Widget" },
    "xpath": {
      "selector": "//item/name"
    }
  }]
}
