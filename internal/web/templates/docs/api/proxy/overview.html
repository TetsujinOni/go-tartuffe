Proxies

Proxies are one of the most powerful features of tartuffe. They allow you to create a test double that records real responses from a downstream system and replays them during subsequent test runs.

Proxy Modes

Tartuffe supports three proxy modes:

Mode Description
proxyOnce Proxies the request on the first call and saves the response. Subsequent calls with matching predicates will return the saved response.
proxyAlways Always proxies the request to the downstream system and saves every response. Useful for recording multiple variations of responses.
proxyTransparent Always proxies the request but does not save responses. Useful for pass-through scenarios where you just want to observe traffic.

Example

The following example creates a proxy that forwards requests to a downstream API:

{
  "port": 4545,
  "protocol": "http",
  "stubs": [{
    "responses": [{
      "proxy": {
        "to": "http://api.example.com",
        "mode": "proxyOnce",
        "predicateGenerators": [{
          "matches": {
            "method": true,
            "path": true,
            "query": true
          }
        }]
      }
    }]
  }]
}

Predicate Generators

Predicate generators tell tartuffe how to create predicates from the proxied request. This determines how subsequent requests will be matched to saved responses.
